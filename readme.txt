[https://www.calculator.io/density-calculator/](https://www.calculator.io/density-calculator/)

== Usage ==

Add the Density Calculator shortcode to your page, post or sidebar:

`[ci_density_calculator]`

Add the following code to your template where you would like the Density Calculator to appear:

`<?php display_ci_density_calculator(); ?>`
